## 🚀 Featured Project: Autonomous CX Analyst

**A dual-track RAG system designed for enterprise NPS analysis.**

- **Reasoning:** Agentic synthesis using Gemini 2.5 Flash to identify systemic operational facts (n=counts).
- **Interface:** Slack-native stakeholder bot with threaded memory.

## 🧠 Featured Project: Life OS

**A self-hosted personal dashboard — expenses captured automatically, health synced from Garmin, AI delivering weekly insights to Telegram.**

- **Automation:** Revolut tap-to-pay → iOS Shortcut → n8n → Claude Haiku categorises → Supabase → balance updated instantly.
- **Intelligence:** Weekly AI spending report generated by Claude, delivered every Sunday morning.
- **Dashboard:** Net worth, budget progress, savings goal ETAs, health metrics and workout log — all in one mobile-first app.

[🔗 View Repository](https://github.com/casanovaedu/life-os)
